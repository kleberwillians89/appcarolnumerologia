# Templates de Imagens para PDF "Mapa da Alma"

Esta pasta contém as imagens de referência e backgrounds usadas na geração do PDF "Mapa da Alma" com 9 páginas.

## Arquivos necessários:

### Imagens principais (OBRIGATÓRIAS):
- `pg1_capa.png` - Capa personalizada (página 1) - Imagem integral
- `bg_2_Pdf.png` - Background padrão para páginas 2, 3, 4, 6, 7, 8 e 9
- `pg5_jornada.png` - Imagem integral para página 5 (Jornada)

### Imagens de referência visual (OPCIONAIS - apenas para design):
- `pag2_numeros.png` - Referência visual da página 2 (Números)
- `pg3_anoPessoal.png` - Referência visual da página 3 (Ano Pessoal)
- `pg4_calendario.png` - Referência visual da página 4 (Calendário)
- `pg6_ciclos.png` - Referência visual da página 6 (Ciclos)
- `pg7_desafios.png` - Referência visual da página 7 (Desafios)
- `pg8_presentes.png` - Referência visual da página 8 (Presentes)
- `pg9_compatibilidade.png` - Referência visual da página 9 (Compatibilidade)

## Formato recomendado:
- **Formato**: PNG (com transparência se necessário)
- **Dimensões**: A4 (595 x 842 pt) ou (210mm x 297mm)
- **Resolução**: 72-150 dpi para web, 300 dpi para impressão
- **Qualidade**: Alta resolução

## Estrutura do PDF:

1. **Página 1**: Capa (pg1_capa.png - imagem integral)
2. **Página 2**: Números (bg_2_Pdf.png + 6 cards em grid 2x3)
3. **Página 3**: Ano Pessoal (bg_2_Pdf.png + timeline + interpretação)
4. **Página 4**: Calendário (bg_2_Pdf.png + 12 meses em grid 3x4)
5. **Página 5**: Jornada (pg5_jornada.png - imagem integral)
6. **Página 6**: Ciclos (bg_2_Pdf.png + 3 cards verticais)
7. **Página 7**: Desafios (bg_2_Pdf.png + 2 cards + desafio maior)
8. **Página 8**: Presentes (bg_2_Pdf.png + 4 cards em grid 2x2)
9. **Página 9**: Compatibilidade (bg_2_Pdf.png + barras de progresso)

## Notas importantes:
- As imagens pg1_capa.png e pg5_jornada.png são usadas como páginas inteiras (fit: cover)
- O background bg_2_Pdf.png é reutilizado em múltiplas páginas
- As imagens de referência (pag2_numeros.png, etc.) servem apenas como guia visual para o layout
